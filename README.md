The user specifies the length of the transform (containing only the factors 2,3,5) - the number of signal samples that will be fed 
to the Fourier transform input. The input values (the number of frequency samples) are generated randomly, and the code outputs the 
complex spectrum of the signal (after the direct Fourier transform) and the reconstructed time signal (after the inverse transform). 
It also displays the maximum discrepancy (max error) between the original signal and the signal obtained after the direct and 
inversetransforms to demonstrate the accuracy of the code.
